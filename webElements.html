HTML

The role of html in web development is to format the layout of a web page. This allows for the formatting of text, images, links, resource, etc.

CSS

CSS allows for styling of a webpage. Webpage elements such as layout, font, and color can all be modified within the CSS files.

JavaScript

JavaScript is the scripting language of the web. It allows for programmers to add functionality to their web pages. Form validation and browser detection are some examples of JavaScript's power.
